# EcoTrip Calculator

The **EcoTrip Calculator** simulates the environmental impact of trips, estimating carbon emissions based on:

- Trip distance
- Mode of transport
- Route profile
- Number of passengers
- Round-trip option

## Objective

To help users make more conscious and sustainable mobility decisions through a simple CO2e estimate.

## How to use

1. Open the `index.html` file in your browser.
2. Fill in the trip details.
3. Click **Calculate Impact**.
4. View:
- Estimated emissions in kg CO2e
- Impact rating
- Approximate number of trees needed to offset emissions
- Suggestion for a more sustainable alternative

## Simplified formula

```text
Emissions = Distance x Transport Factor x Route Multiplier
```

For individual vehicles (cars/motorcycles), emissions are divided by the number of passengers.

## Notes

- Emission factors are approximate and intended for educational and comparative purposes.
- The tree calculation (21 kg CO2/year per tree) is a simplified estimate.
